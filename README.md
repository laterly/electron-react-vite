# Electron + React + Vite - Builder

![Electron + React + Vite  ](screenshot.png)

## Quick Start

```sh
# clone the project
git clone https://github.com/laterly/electron-react-vite.git

# enter the project directory
cd electron-react-vite

# install dependency
npm install

# develop
npm run dev

#build

npm run build

#builder

npm run builder
```
